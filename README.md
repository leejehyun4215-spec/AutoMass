# AutoMass

> **Design. Don't Model.**

Automatically generate accurate 3D building models from CAD drawings.

---

## 🚀 Vision

Help architects and urban designers focus on **design**, not repetitive 3D modeling.

---

## 🎯 Mission

Convert CAD drawings into accurate 3D models automatically.

---

## ✨ MVP Features

- Read DWG / DXF
- Detect building outline
- Generate accurate 3D Mass
- Export OBJ
- SketchUp Import
